# Node.js basics

### File system (src/fs)

create:
```sh
npm run fs:create
```

copy:
```sh
npm run fs:copy
```

rename:
```sh
npm run fs:rename
```

delete:
```sh
npm run fs:delete
```

list:
```sh
npm run fs:list
```

read:
```sh
npm run fs:create
```

### Command line interface(src/cli)

env:
```sh
npm run cli:env
```

args:
```sh
npm run cli:args
```

### Modules(src/modules)

modules:
```sh
npm run modules
```

### Hash (src/hash)

hash:
```sh
npm run hash
```

### Streams (src/streams)
read:
```sh
npm run streams:read
```

write: 
```sh
npm run streams:write
```
Type data into console and press ENTER. To stop, press CTRL+C.

transform:
```sh
npm run streams:transform
```
Type data into console and press ENTER. To stop, press CTRL+C.

### Zlib (src/zip)

compress:
```sh
npm run zlib:compress
```

decompress:
```sh
npm run zlib:decompress
```
